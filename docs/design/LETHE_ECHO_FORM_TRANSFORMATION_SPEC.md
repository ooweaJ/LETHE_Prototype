# LETHE 잔향 형태 변환 문법

최종 갱신: 2026-06-10

## 0. 이 문서의 위치

`LETHE_WEAPON_MEMORY_ECHO_DETAIL.md`가 각 기억/잔향의 레벨별 동작을 정리한다면, 이 문서는 그 동작이 화면에서 어떤 형태 변화로 읽혀야 하는지 정의한다.

핵심 문장:

```text
잔향은 약해진 같은 효과가 아니라, 잃어버린 기억이 무기에 남긴 버릇이다.
```

따라서 활성 기억, 일반 잔향, 각성 잔향, 공명, 궁극 잔향은 모두 같은 소재를 쓰더라도 화면 언어가 달라야 한다.

## 1. 형태 변환 문법

| 상태 | 플레이어가 보는 형태 | 의미 |
| --- | --- | --- |
| 활성 기억 | 플레이어 주변/전장에 독립적으로 존재하는 능력 | 지금 내가 가진 힘 |
| 망각 순간 | 독립 형태가 무기/몸 안으로 빨려 들어감 | 힘을 잃었지만 흔적이 남음 |
| 잔향 +1~+3 | 무기 공격 뒤에 따라붙는 작은 사건 | 무기에 남은 버릇 |
| 잔향 +4 | 무기 주변에 희미하게 붙은 반각성 파편 | 이 흔적을 믿을 수 있음 |
| 잔향 +5 | 타격/처치/피격에 반응하는 각성 형태 | 빌드 목표가 열린 상태 |
| 공명 | 활성 기억이 돌아오고 잔향도 유지됨 | 잃었던 힘과 흔적이 동시에 존재 |
| 궁극 잔향 | 두 각성 잔향이 서로를 먹여 살리는 루프 | 내가 잃은 기억들로 만든 빌드 |

금지:

- 잔향 +5를 활성 기억과 똑같은 화면 형태로 만들지 않는다.
- 활성 기억과 잔향의 차이를 텍스트/색상만으로 구분하지 않는다.
- 공명을 단순 레벨 보정으로만 처리하지 않는다.

## 2. 굶주린 칼무리 예시

### 활성 기억

- 플레이어 주변을 독립 칼날 고리가 돈다.
- 무기 공격과 무관하게 가까운 적을 자동으로 벤다.
- 플레이어는 "내 주변을 도는 능력"으로 인식한다.

### 망각 순간

- 칼날 고리가 흔들리고, 칼날들이 플레이어 무기 쪽으로 빨려 들어간다.
- 고리가 사라지는 직후 무기 날에 희미한 잔광이 남는다.
- UI 문구보다 이 장면이 먼저 "기억을 잃었지만 무기에 남았다"를 설명해야 한다.

### 칼무리 잔향 +1~+3

- 쌍검 타격 뒤에 반달 칼자국이 한 박자 늦게 지나간다.
- 칼자국은 플레이어 주변 고리가 아니라, 무기 궤적에서 튀어나온다.
- 레벨이 오르면 빈도, 체류 시간, 맞는 적 수가 늘어난다.

### 칼무리 잔향 +4

- 완전한 고리가 아니라 손/무기 주변에 희미한 칼날 파편 1개가 붙는다.
- 평상시에는 약하고, 타격 순간에만 선명해진다.
- "활성 기억이 돌아왔다"가 아니라 "무기에 칼무리 버릇이 붙었다"로 보여야 한다.

### 칼무리 잔향 +5

- 반각성 파편 고리가 유지된다.
- 고리는 독립 자동 공격보다 타격/처치에 반응한다.
- 타격 시 고리에서 칼날 1개가 가까운 적에게 발사되고, 처치 시 짧게 가속한다.

### 공명

- 굶주린 칼무리를 다시 얻으면 독립 칼날 고리가 돌아온다.
- 잔향 칼선과 활성 고리가 둘 다 남는다.
- 활성 칼날이 맞힌 적을 무기로 치면 잔향 칼선이 추가로 터진다.

### 피의 칼폭풍

- 칼날 고리와 혈반이 합쳐져 붉은 칼날 폭풍이 된다.
- 칼날이 적에게 혈반을 묻히고, 혈반이 터지면 회복 실이 돌아온다.
- 처치가 이어지면 고리가 가속하고 사운드 레이어가 차오른다.

## 3. 피의 반사 예시

### 활성 기억

- 기본 공격에 피의 추가타와 작은 회복이 붙는다.
- 플레이어는 "때리면 피가 돌아온다"로 인식한다.

### 혈반 잔향

- 적에게 붉은 혈반이 묻고, 다시 치면 실처럼 플레이어에게 돌아온다.
- 레벨이 오르면 회복 실의 수, 전파, 피꽃 폭발이 커진다.
- 혈반은 칼무리처럼 공간을 점유하지 않고, 적 몸에 남는 흔적이어야 한다.

### 궁극 융합

- 칼무리 칼날이 혈반을 묻히고, 혈반이 터지면 회복 실이 돌아온다.
- 두 효과는 순차 발동이 아니라 서로를 먹여 살리는 루프로 보여야 한다.

## 4. 이벤트 경계

공명과 궁극은 강하지만, 무한 발동 루프를 만들면 안 된다.

| 이벤트 | 잔향 발동 가능 여부 | 용도 |
| --- | --- | --- |
| `WeaponHit` | 가능 | 무기 기본 타격. 대부분 온힛 잔향의 시작점 |
| `EchoHit` | 제한 | 잔향이 만든 타격. 일반 잔향 재발동 금지, 공명 rider만 허용 |
| `UltimateHit` | 제한 | 궁극 잔향 타격. 궁극 전용 전파/회복만 허용 |
| `Kill` | 가능 | 처치 가속, 전파, 처형 연쇄 |
| `DamageTaken` | 제한 | 잿빛 보호막 계열 |
| `ShieldBreak` | 가능 | 잿빛 각성, 성채 파문 |

규칙:

- `WeaponHit -> EchoHit -> EchoHit` 무한 체인을 금지한다.
- `EchoHit`는 `WeaponHit`처럼 모든 온힛 잔향을 다시 호출하지 않는다.
- 궁극 잔향은 자체 쿨다운, 초당 회복량 cap, 타겟 cap을 가진다.
- 처치/전파는 강하게 보여도 한 프레임 안에서 재귀적으로 폭발하지 않는다.

## 5. 타격 피드백 우선순위

Unity에서 잔향 뽕맛을 만드는 순서는 아래가 우선이다.

1. 히트스톱: 각성/궁극/처형 발동 순간 3~5프레임 정지.
2. 스프라이트 화이트 플래시: 맞은 적 1프레임 흰색.
3. 사운드 레이어링: 일반 타격, 잔향, 각성, 궁극 소리가 쌓임.
4. 칼선/피실/파편 파티클: 형태 변환을 직접 보여줌.
5. 카메라 임펄스: 궁극/처형/강타에만 제한적으로 사용.
6. 2D 라이트/발광: +5 각성과 궁극에서만 선명하게 사용.
7. 오브젝트 풀링: 체감 효과는 아니지만 밀도 높은 잔향을 안전하게 유지하는 필수 기술.

## 6. 첫 쇼케이스 완료 기준

대상:

```text
절단쌍검 + 굶주린 칼무리 + 피의 반사 -> 피의 칼폭풍
```

완료 기준:

- 망각 순간에 칼무리 고리가 무기 쪽으로 빨려 들어가는 장면이 보인다.
- 망각 후 10초 안에 플레이어가 무기 공격에 남은 칼선 변화를 본다.
- 칼무리 잔향 +5는 +1~+3과 다른 행동을 가진다.
- 혈반 잔향 +5는 회복 실과 피꽃으로 생존 루프를 만든다.
- 피의 칼폭풍은 칼날이 혈반을 묻히고 혈반이 회복 실을 돌려주는 루프로 보인다.
- 공명 시 활성 고리와 무기 칼선이 동시에 존재한다.
- 플레이어가 "잃은 기억들이 내 무기에 남아 빌드가 됐다"라고 설명할 수 있다.

## 7. 구현 판단

HTML에서 증명할 수 있는 것:

- 망각 전후 규칙.
- 잔향 레벨, HUD, 궁극 목표.
- 간단한 칼선/혈반/회복 루프.

Unity에서 증명해야 하는 것:

- 히트스톱, 사운드, 라이트, 파티클로 만드는 타격감.
- 고리에서 무기로 빨려 들어가는 망각 장면.
- 각성/궁극 발동의 순간적인 화면 장악력.
- 오브젝트 풀 기반으로 밀도 높은 잔향을 유지하는 안정성.

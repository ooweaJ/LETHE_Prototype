# LETHE Unity Vertical Slice 명세서

최종 갱신: 2026-06-09

## 목적

이 문서는 Unity에서 가장 먼저 만들 범위를 정의한다.

목표는 전체 게임을 만드는 것이 아니라, 기억/잔향/공명/각성/궁극 잔향 루프가 재미있는지 작게 증명하는 것이다.

## 첫 slice 목표

플레이어가 한 번의 짧은 플레이 안에서 다음을 경험해야 한다.

1. 기억을 얻고 강화한다.
2. 기억을 잃는다.
3. 잃은 기억이 잔향으로 남는다.
4. 잔향이 남은 기억을 다시 얻어 공명을 본다.
5. 잔향 `+5` 각성을 본다.
6. 가능하면 궁극 잔향 1개를 본다.

## 구현 범위

### 필수

- 플레이어 이동.
- 기본 무기 공격.
- 근접 적 1종.
- 원거리 적 1종.
- 문지기 또는 엘리트 1종.
- 활성 기억 슬롯.
- 기억 레벨 `+1~+5`.
- 망각 이벤트.
- 잔향 레벨 `+1~+5`.
- 잔향 초과분 과부하 처리.
- 재획득 공명 규칙.
- 각성 잔향 2개.
- 궁극 잔향 1개.
- 현재 기억/잔향/시너지 debug UI.
- 루프 강제 시연 debug UI: 즉시 망각, 잔향 `+5`, 궁극 잔향 발동.

### 보류

- 상점.
- 메타 성장.
- 다중 지역.
- 최종보스.
- 대규모 기억 확장.
- 대규모 서사.
- 저장/로드.

## 추천 첫 쇼케이스

첫 구현 조합:

- 기억 1: 굶주린 칼무리.
- 기억 2: 피의 반사.
- 각성 잔향 1: 칼무리 잔향.
- 각성 잔향 2: 혈반 잔향.
- 궁극 잔향: 피의 칼폭풍.

이 조합은 주변 칼날, 온힛, 출혈, 피의 추가타가 한 화면에서 보이므로 잔향 시스템을 설명하기 좋다.

## 완료 기준

- 기억을 잃으면 잔향 레벨이 오른다.
- 망각 대상은 최고 레벨 활성 기억이며, 동률이면 플레이어가 선택한다.
- `+3` 잔향이 있는 상태에서 `+2` 기억을 잃으면 `+5` 잔향이 된다.
- 잔향 `+5`를 넘는 초과분은 폐기되지 않고 과부하 보상으로 전환된다.
- 잔향이 있는 기억을 다시 얻으면 기억 레벨 또는 효과가 강화된다.
- `+5` 잔향은 화면에서 확실히 보인다.
- 궁극 잔향은 일반 잔향보다 강한 보상으로 느껴진다.
- 원거리 적은 계속 도망가지 않는다.

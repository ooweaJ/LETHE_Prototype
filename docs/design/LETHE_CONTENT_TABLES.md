# LETHE 콘텐츠 표

최종 갱신: 2026-06-10

## 목적

이 문서는 기억, 잔향, 각성 잔향, 궁극 잔향, 적 역할을 표로 관리한다.

구체적인 무기별 행동, 공명 라이더, 궁극 잔향의 화면 연출은 `LETHE_WEAPON_MEMORY_ECHO_SPEC.md`를 기준으로 한다.

## 기억과 잔향

| 기억 | 태그 | 활성 기억 방향 | 잔향 방향 | 각성 잔향 |
| --- | --- | --- | --- | --- |
| 굶주린 칼무리 | 근접 / 지속 / 온힛 | 주변 적을 계속 베는 칼날 | 무기 타격에 칼날 잔흔 | 칼무리 잔향 |
| 처형자의 섬광 | 폭딜 / 치명 | 위협 적에게 큰 섬광 피해 | 치명/처형 보정 | 처형 잔향 |
| 추적자의 맹세 | 원거리 / 추적 | 유도 투사체 | 추적 잔탄 | 추적 잔향 |
| 파쇄의 파문 | 광역 / 제어 | 충격파와 넉백 | 작은 파문 | 파문 잔향 |
| 피의 반사 | 온힛 / 회복 | 기본 공격 추가타 | 피의 잔타 | 혈반 잔향 |
| 멈춘 초침 | 제어 / 쿨다운 | 둔화장 또는 시간 제어 | 짧은 둔화 | 정지 잔향 |
| 잿빛 보호막 | 방어 / 반격 | 보호막 | 방어 반응 | 잿빛 잔향 |
| 망각의 낙인 | 표식 / 폭딜 | 취약 표식 | 짧은 낙인 | 낙인 잔향 |

## 궁극 잔향 후보

| 필요 각성 잔향 | 궁극 잔향 | 핵심 효과 |
| --- | --- | --- |
| 칼무리 잔향 + 혈반 잔향 | 피의 칼폭풍 | 주변 칼날이 출혈과 피의 추가타를 만든다. |
| 처형 잔향 + 낙인 잔향 | 처형 각인 | 낙인 적 처형 시 하얀 폭발이 연쇄된다. |
| 추적 잔향 + 정지 잔향 | 정지 추적 | 유도 잔탄이 둔화 적에게 연쇄된다. |
| 파문 잔향 + 잿빛 잔향 | 성채 파문 | 방어 반응이 큰 넉백 파문을 만든다. |

## Unity 첫 slice 추천 후보

1. 굶주린 칼무리.
2. 피의 반사.
3. 칼무리 잔향.
4. 혈반 잔향.
5. 궁극 잔향: 피의 칼폭풍.

이 조합은 온힛, 주변 베기, 출혈, 추가타가 한 화면에서 잘 보이기 때문에 첫 쇼케이스로 적합하다.

## 적 표

| 적 | 역할 | Unity 방향 |
| --- | --- | --- |
| 침식자 | 기본 근접 적 | 다수로 몰려와 처치 쾌감을 만든다. |
| 떠도는 눈 | 원거리 압박 적 | 짧게 물러나고 멈춰서 쏜 뒤 다시 압박권으로 복귀한다. |
| 쪼개진 자 | 분열 적 | 광역/잔향 효과의 가치를 보여준다. |
| 공허 사제 | 지원 적 | 우선 처치 판단을 만들되 첫 slice에서는 보류 가능. |
| 문지기/엘리트 | 빌드 시험 | 첫 망각 전후의 빌드 강도를 확인한다. |

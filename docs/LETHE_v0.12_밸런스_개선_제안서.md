# LETHE v0.12 밸런스 개선 제안서

기준 문서: `BALANCE_TABLE_v0_11.md`  
작성일: 2026-06-04

## 핵심 결론

판정: `FIX_BALANCE_BASELINE_BEFORE_CONTENT_EXPANSION`

현재 v0.11 수치는 기능 확인용 임시값에 가깝고, 실제 플레이 밸런스로는 초반 공백과 후반 폭주가 동시에 발생할 위험이 크다. v0.12는 콘텐츠 추가보다 DPS 기준화, 보상 리듬 재설계, Proc 상한, 망각 납득성 개선을 우선한다.

## 1. 현재 밸런스 진단

| 우선순위 | 항목 | 현재 문제 | v0.12 판단 |
|---:|---|---|---|
| 1 | 굶주린 칼무리 | `dt` 없이 프레임마다 7.4 피해. 60fps 기준 단일 약 444 DPS 가능 | 버그 수정급. 초당 피해 구조로 즉시 변경 |
| 2 | 피의 늪 | 쌍검 42% 발동, 0.34초 틱, 2.6초 지속으로 화면 장악 가능 | 발동률, 틱, 지속시간, 동시 개수 제한 필요 |
| 3 | 첫 보스 HP | 현재 381. 기본 무기 + 기억 DPS 기준 너무 빨리 녹을 수 있음 | 목표 TTK 기준으로 900~1,200 재산정 |

| 5 | 망각 확률 | 전투 의존 72%라 사실상 딜 1위 삭제처럼 보일 위험 | 상위 1~2개 후보 중심, 보스/생존 기여 반영 |
| 6 | 스폰 리듬 | 스폰 간격만 있고 초당 예산/최대 생존 적 수가 없음 | 스폰 예산표와 객체 상한 필요 |

## 5. 런 성장 선택지 조정

| 선택 | 현재 | 수정 추천 | 비고 |
|---|---|---|---|
| 칼날 가속 | 공속 +11%, 기억 쿨감 +4% | 공속 +8%, 기억 쿨감 +2.5~3% | 공속과 쿨감은 둘 다 DPS 계열 |
| 검은 물의 힘 | 피해 +14% | 피해 +10~12% | 가장 단순하고 강한 선택 |
| 파문 확장 | 범위 +12%, 넉백 +8% | 범위 +9~10%, 넉백 +6% | 광역 기억과 피의 늪 폭주 방지 |
| 가라앉지 않는 숨 | HP +12, 즉시 +18, 피해감소 +4% | HP +10, 즉시 +14, 피해감소 +3% | 생존 선택은 유지하되 과한 안정성 방지 |
| 기억 흡입 | XP +16% | XP +12~14% | 초반 보상 리듬 로그 보고 조정 |
| 잔향 증폭 | 잔향 +20% | 잔향 +10~12%, 최대 3스택 | 후반 폭주 핵심 레버 |

권장 피해식:

```text
finalDamage = baseDamage * (1 + generalDamageSum) * (1 + roleDamageSum) * synergyMultiplier
synergyMultiplier <= 1.35~1.45
```

## 14. 최종 정리

LETHE는 빠르게 강해지는 오토배틀 로그라이크의 쾌감을 가져가야 하지만, 그 강함이 끝까지 보존되는 게임은 아니다. v0.12의 핵심은 “강해짐 → 완성 → 망각 → 결손 생존 → 잔향 피벗”이 10분 압축 런에서 실제로 재미있게 반복되는지 확인하는 것이다.

# 2026-06-27 Balance Simulation V1

## 목적

Unity `Dev_Prototype_v1`을 20분 베타 런으로 조정하기 위해 XP, 문지기 스케줄, 보스 HP, 궁극 완성 타이밍을 비교했다.

## 실행

```powershell
node scripts\balance_sim_v1.js
```

## 판정

- 적용: 시작 XP 7, 0~120초 XP ×1.00, 120~600초 XP ×1.34, 초반 처치 XP 보너스 제거.
- 적용: 런 하드 캡 1260초, 문지기 300 / 600 / 900 / 1140초, 문지기 HP 1900 / 2800 / 4000 / 5400.
- 적용: 1260초 생존만으로 승리하지 않고, 4번째 문지기 처치를 클리어 조건으로 둔다.
- 적용: 레벨업 보상 우선순위를 피의 칼폭풍 전용에서 4개 궁극 조합 공통으로 확장한다.

## 남은 검증

- 순수 수치 시뮬레이션은 위치, 피격, 회피, VFX 가림, 실제 조작감을 반영하지 않는다.
- 대검 루트의 클리어율이 쌍검보다 낮게 나오므로 MCP 장기 플레이와 jaewoo 실플레이에서 첫 번째 확인 대상이다.
- 첫 120초 레벨 3.5~4.0이 너무 느리게 느껴지면 0~120초 XP를 1.06까지 올리는 `20m_target` 계열을 재검토한다.

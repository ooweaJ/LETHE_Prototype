Codex 앱과의 차별점

이 오케스트레이션은 대화창 안에만 남는 상태가 아니라, 개발 문서와 보고서를 프로젝트 폴더에서 직접 볼 수 있게 남깁니다.

사용자는 HTML 대시보드와 보고서 목록을 보고 다음 프롬프트를 직접 결정할 수 있습니다.

Markdown이 원본입니다. 이 HTML 3종(`index.html`, `command.html`, `runbook.html`)은 사람이 보는 오케스트레이션 인터페이스이며, `reports/`와 `devlog/` 목록은 개발 문서를 직접 훑기 위한 보조 화면입니다.

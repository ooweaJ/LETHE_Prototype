# Project Brief

## One-Line Summary

LETHE is an HTML prototype for testing whether a memory-loss roguelike loop is fun and promising enough to justify a later Unity implementation.

## Current Goal

Concretize how weapons, memories, echoes, resonance, and ultimate echoes should feel before doing more implementation. The current focus is not content expansion; it is turning abstract echo labels into concrete combat events.

## Current Phase

Design concretization after HTML `ITERATE_BEFORE_TEST` revealed that the new rules do not yet feel big enough.

## Tech Stack

- Static HTML/CSS/JavaScript prototype.
- Node.js scripts for QA, reporting, packaging, and planning support.
- Browser/Chrome-based automation for balance and flow gates.
- Markdown reports generated into HTML for readable sharing.

## Portfolio Framing

LETHE is a portfolio case study for AI-assisted prototyping and evidence-driven iteration:

- Problem: verify a risky memory-loss core loop before full production.
- Direction: build an HTML slice that can expose emotion, readability, and balance risks early.
- Action: use Codex for implementation, QA, reporting, and records; use Claude/GPT-style review for planning interpretation.
- Result target: human playtest evidence that supports either Unity transition or another focused HTML iteration.

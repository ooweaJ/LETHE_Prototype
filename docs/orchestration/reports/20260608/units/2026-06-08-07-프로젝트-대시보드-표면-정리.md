- 대시보드에서 설명성 문구와 하단 링크 카드를 제거했다.

## 2. 오늘 바뀐 것

- 제목을 `상태 요약`으로 단순화했다.

## 3. 테스트 결과와 근거

- `npm run report`: 통과.

- `npm run doctor`: 통과.
- 문자열 확인으로 제거 대상 문구가 남지 않았음을 확인했다.

## 5. 문제 또는 리스크

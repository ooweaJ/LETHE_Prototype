1. 현재 빌드 상태

Dev_Prototype_v1은 계속 _dev 검증 빌드다. 다만 이번 작업부터는 작은 테스트 박스 느낌을 줄이는 쪽으로 방향을 바꿨고, 맵/배경을 실제 출시 준비용 월드 베이스에 가깝게 키우기 시작했다.

2. 오늘 바뀐 것

  • 새 맵 스프라이트를 추가했다:
  • tile_lethe_stone_01.png
  • tile_lethe_stone_02.png
  • tile_lethe_stone_03.png
  • tile_lethe_stone_04.png
  • spr_lethe_arena_backdrop_01.png
  • 같은 이미지를 다시 만들 수 있게 scripts/generate_world_sprites.ps1를 추가했다.
  • 플레이어 이동 범위를 기존 x +/-12, y -8.5..8.5에서 x +/-24, y +/-16으로 키웠다.
  • 바닥 타일 배치도 11x9에서 21x15로 늘렸다.
  • 카메라 크기를 6.1에서 6.8로 키우고, 커진 맵 안에서만 따라가도록 클램프했다.
  • 적 스폰 반경을 늘려서 커진 맵에서 바로 몸에 붙는 느낌을 줄였다.
  • 배경, 경계선, 균열, 외곽 표식을 더 큰 맵 기준으로 다시 배치했다.

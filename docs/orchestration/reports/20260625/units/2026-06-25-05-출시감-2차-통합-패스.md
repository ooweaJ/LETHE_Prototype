# 2026-06-25-05 - 출시감 2차 통합 패스

## 1. 현재 빌드 상태

`Dev_Prototype_v1`은 이제 대검/쌍검 전투감 1차 정리 위에 맵 연결감, 적 판독, 기억/잔향 밸류, 성장 피드백을 추가로 얹은 상태다. 아직 정식 출시 구조는 아니고 `_dev` 검증 빌드지만, “프로토타입 박스”에서 “출시 준비용 게임 셸” 쪽으로 더 가까워졌다.

## 2. 오늘 바뀐 것

- 맵에 레테 강줄기, 강둑, 가라앉은 폐허 판을 런타임으로 추가했다.
- 모든 적/보스에게 HP 바를 붙였다.
- 혈반 표식이 풀릴 때 적 색이 흰색으로 돌아가는 문제를 고치고, 원래 적 색으로 복구되게 했다.
- 처형 섬광, 추적자의 맹세, 파쇄의 파문, 멈춘 1초, 잿빛 방패, 낙인 계열의 수치와 VFX를 키웠다.
- 정지 추적, 파쇄 처형, 잿빛 망각 같은 유틸 궁극 잔향의 범위/피해/정지/회복 값을 올렸다.
- 레벨업 순간에 청록/금색 버스트가 나오게 했다.
- 궁극 조건이 준비되면 색상 버스트, 텍스트, 짧은 카메라 흔들림이 나오게 했다.

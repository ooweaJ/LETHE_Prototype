# 2026-06-25-02 - 통합 리뷰 프리셋 추가

## 1. 현재 빌드 상태

대검 VFX는 보장형 cleave로 다시 보이게 만들었지만, 이제 남은 문제는 직접 리뷰를 빠르게 반복하는 것이다. 기존에는 대검/쌍검, 잔향 과밀, 적 판독, 문지기 예고를 보려면 플레이를 길게 진행해야 했다.

## 2. 오늘 바뀐 것

- 디버그 패널에 `DB Rev`, `GS Rev` 버튼을 추가했다.
- 버튼을 누르면 바로 통합 리뷰 상태로 들어간다:
  - 필요한 경우 런 시작
  - 선택 무기로 전환
  - 주변에 리뷰용 적 18마리 배치
  - 8잔향 전체 +5 설정
  - 궁극은 Echo Only 모드로 억제
  - 문지기 예고 타이밍을 20초로 설정
- 디버그 패널 높이를 늘려 버튼과 로그가 겹치지 않게 했다.

## 3. 테스트 결과와 근거

- `dotnet build LETHE/Assembly-CSharp.csproj --nologo`: 통과, warning 0개, error 0개.
- Unity `Assets/Refresh`: 성공.
- Unity compile error count: `0`.
- Unity Play Mode 진입: `isPlaying=true` 확인.
- Unity console에는 MCP 서버 정보 로그만 있었고 게임 런타임 오류/경고는 없었다.

## 4. 결정한 것

지금 단계에서는 감각 판단을 빨리 반복하는 도구가 필요하다. `DB Rev`와 `GS Rev`는 일반 플레이를 대체하는 모드가 아니라, 무기 VFX/잔향 과밀/적 판독/문지기 예고를 한 화면에서 빠르게 확인하기 위한 리뷰 프리셋이다.

## 5. 문제 또는 리스크

리뷰 프리셋은 의도적으로 과밀한 상황을 만든다. 실제 150초 플레이보다 더 복잡하게 보일 수 있으니, 이 버튼에서 문제를 찾은 뒤 일반 플레이에서도 같은 문제가 반복되는지 확인해야 한다.

## 6. GPT/Claude 인계 요약

## 4. 결정한 것

- 기본공격은 게임을 굴리는 베이스일 뿐, 빌드 정체성은 기억/잔향이 가져야 한다.
- 칼무리 = cyan orbit blade / nearby cuts.
- 피의 반사 = red mark / heal thread / heal bloom.
- 스프라이트 최종 퀄리티는 별도 패스가 필요하지만, 먼저 효과의 역할과 판독성을 잡는다.

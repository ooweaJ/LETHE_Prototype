# 2026-07-23-04 - 낙인 머리표식 / 잿빛 성화 영역 후속 패치

## 1. 현재 빌드 상태

`Dev_Prototype_v1`에서 낙인은 몬스터 머리 위에 전용 표식이 뜨고, 잿빛 기억은 적 주변에 잿빛 성화 영역을 만들어 도트 피해를 주는 구조로 바뀌었다.

## 2. 오늘 바뀐 것

- 낙인:
  - 기억 새김, 잔향 새김, 지연 삭제, 전염 시점에 몬스터 머리 위 전용 낙인 표식을 생성한다.
  - 기존 상태 배지보다 더 위쪽에, 더 명확한 망각 문양으로 표시한다.
- 잿빛:
  - 기억이 위협 대상 근처에 `잿빛 성화 영역`을 만든다.
  - 영역은 잠깐 유지되고, 안에 들어온 적에게 도트 피해를 준다.
  - 영역 피해는 소량의 방어 저장량도 쌓아 잿빛의 방어/반격 성격과 이어진다.
  - 기존 방패/반격 구조는 유지하되, 공격 정체성은 공간 점유형으로 강화했다.

## 4. 결정한 것

낙인은 피격 순간보다 먼저 “이 몬스터에게 낙인이 찍혔다”는 머리 위 표식이 있어야 한다. 잿빛은 단순 자동 반격보다 `잿빛 영역`이라는 공간 정체성을 가져야 한다.

## 5. 문제 또는 리스크

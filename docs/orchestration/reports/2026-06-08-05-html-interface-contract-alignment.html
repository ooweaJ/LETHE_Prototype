최신 오케스트레이션 보고서

현재 빌드 상태

LETHE v0.12는 여전히 controlled human-test candidate입니다. 이번 작업은 gameplay 변경 없이 오케스트레이션 HTML 인터페이스와 문서 규약을 맞춘 작업입니다.

무엇을 바꿨나

  • index.html을 한국어 사람용 대시보드로 복구했습니다.
  • command.html을 다음 지시 블록으로 정리했습니다.
  • runbook.html을 반복 운영 절차 블록으로 정리했습니다.
  • reports/ 안에서 사람용 보고서 HTML을 직접 열 수 있게 했습니다.

검증

  • npm run report: pass, 5 unit reports generated
  • npm run report:check: pass
  • npm run report:discord:unit:dry: pass
  • npm run doctor: 50 pass / 0 warn / 0 fail

결정

필수 HTML 인터페이스는 index.html, command.html, runbook.html입니다. reports/index.html과 devlog/index.html는 문서 탐색을 위한 보조 화면으로 둡니다.

리스크

  • HTML은 아직 수동 유지라 Markdown과 어긋날 수 있습니다.
  • 여러 프로젝트에 적용하려면 Markdown 기반 생성기가 필요합니다.

다음 작업

  • 사람 테스트 세션을 진행합니다.
  • JSON 로그를 playtest_logs/에 저장합니다.
  • npm run playtest:summary로 요약합니다.

포트폴리오 메모

Codex 대화와 별개로 프로젝트 폴더 안에서 현재 상태, 다음 지시, 운영 절차, 보고서를 볼 수 있게 만든 점이 차별점입니다.

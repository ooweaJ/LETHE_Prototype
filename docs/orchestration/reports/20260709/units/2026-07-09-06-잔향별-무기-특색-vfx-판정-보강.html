1. 현재 빌드 상태

  • Assets/_dev의 Dev_Prototype_v1에서 일반 잔향들도 무기별 VFX와 판정 보상이 더 분리된 상태다.
  • 쌍검 밀집 상황은 중간에 예산을 넘었지만, 최종 분기는 무거운 보조 이펙트를 억제해서 다시 PASS 상태로 돌려놨다.

2. 오늘 바뀐 것

  • 혈반, 처형, 사냥, 파쇄, 멈춘 초침, 잿빛 방패, 망각 낙인 잔향에 무기별 보조 VFX를 추가했다.
  • 대검은 축을 가르는 흡혈, 단두대 판결, 창 그림자 추적, 균열선, 시계바늘 절단, 방벽 균열, 붕괴 고리처럼 더 큰 한 방으로 읽히게 했다.
  • 쌍검은 봉합 절단, 연쇄 표식, 부채꼴 바늘, 타겟 핍, 틱 컷, 패링 불꽃, 중첩 고리처럼 빠르게 쌓이는 표식으로 읽히게 했다.
  • 사냥, 파쇄, 멈춘 초침, 잿빛 방패, 망각 낙인의 발동률/범위/피해/제어값을 조정해서 약하거나 수동적인 기억도 잔향 단계에서 보상이 생기게 했다.

4. 결정한 것

  • 약한 기억은 약한 잔향으로 끝나면 선택지가 죽으므로, 잔향/궁극 단계에서 더 강한 변형과 보상을 주는 방향을 유지한다.
  • 대검은 크고 느린 판결, 쌍검은 빠르게 누적되는 표식이라는 무기 문법을 다른 잔향에도 적용한다.
  • 밀집 쌍검은 모든 보조 VFX를 다 보여주기보다 핵심 피드백을 남기고 비싼 보조 효과를 줄이는 쪽이 맞다.

5. 문제 또는 리스크

  • 자동 QA는 개수, 예산, 발동 범위를 보지만 실제 쾌감은 직접 플레이 판단이 필요하다.
  • 잔향별 특색이 많아진 만큼 일부 효과는 화면에서 너무 바쁘게 느껴질 수 있다.
  • 대검 쪽은 더 커졌지만, 실제 히트 타이밍과 VFX 타이밍이 어긋나면 무거움보다 둔함으로 느껴질 수 있다.

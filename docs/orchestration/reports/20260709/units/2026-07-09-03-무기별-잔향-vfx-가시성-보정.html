2026-07-09-03 - 무기별 잔향 VFX 가시성 보정

1. 현재 빌드 상태

Dev_Prototype_v1에서 잔향 VFX의 무기별 크기와 가시성을 1차 보정했다. 아직 _dev 검증 단계이며 Assets/Lethe 승격은 하지 않았다.

2. 오늘 바뀐 것

  • 칼무리, 잔향, 궁극 잔향 계열 VFX가 일반 무기 슬래시보다 위에 보이도록 정렬 순서를 올렸다.
  • 칼무리 잔향의 푸른 균열, 칼날 끌림, 물림 자국, 트레일 밝기를 키웠다.
  • 쌍검 dense 상황에서는 성능 예산을 넘지 않도록 칼무리 조각 수를 줄이되, 푸른 펄스와 흉터는 남기도록 했다.
  • 대검 잔향은 더 큰 도장, 파열, 돔, 실드, 폭발, 링 중심으로 키웠다.
  • 쌍검 잔향은 더 밝은 중첩 베기, 짧은 링크, 반복 표식 중심으로 키웠다.
  • 혈반, 처형, 추적, 파쇄, 정지, 잿빛, 낙인 잔향 모두 대검/쌍검 차이가 더 드러나도록 조정했다.

3. 테스트 결과와 근거

  • Unity compilation errors: 0.
  • Kalmuri Perf Matrix: PASS, totalKalmuri=420.
  • Dense Dual Blades Perf Matrix: PASS, hits=18, suppressed=15, transient=156, activeVfx=73, ms=99.50.
  • Echo Matrix Dual Blades: PASS, total=232, K=8, state=87.
  • Echo Matrix Greatsword: PASS, total=231, K=8, state=59.

4. 결정한 것

잔향을 단순히 전부 크게 키우지 않고, 대검은 큰 단발 충격과 넓은 판정으로, 쌍검은 빠른 중첩 흔적과 짧은 표식으로 나눴다. 칼무리는 무기 슬래시에 가려지지 않도록 레이어와 푸른 칼날 흔적을 우선 보정했다.

5. 문제 또는 리스크

  • Kalmuri Perf Matrix가 totalKalmuri=420으로 현재 안전선에 딱 맞는다. 더 키우려면 오브젝트 수보다 스케일, 알파, 수명 위주로 조정해야 한다.
  • 자동 QA는 생성 수와 예산만 보장한다. 실제로 대검이 충분히 묵직한지, 쌍검이 너무 시끄러워지지 않았는지는 직접 플레이 판단이 필요하다.

6. GPT/Claude 인계 요약

이번 변경은 “대검은 크게, 쌍검은 선명하게, 칼무리는 위 레이어에서 보이게”가 핵심이다. 다음 리뷰는 특정 잔향 가족 단위로 너무 작거나 너무 시끄러운 것을 골라야 한다.

7. 다음 Codex 작업

jaewoo가 두 무기로 직접 플레이하면서 칼무리, 혈반, 처형, 추적, 파쇄, 정지, 잿빛, 낙인을 비교한다. 이후 약한 계열 하나씩만 추가 튜닝한다.

8. 포트폴리오 메모: 문제, 방향, 행동, 결과

  • 문제: 잔향 VFX가 무기 이펙트에 묻히고, 대검/쌍검의 잔향 개성이 충분히 갈라지지 않았다.
  • 방향: 전역 확대가 아니라 무기별 VFX 문법을 분리했다.
  • 행동: 정렬 순서, 칼무리 스케일/알파, 대검 충격형 VFX, 쌍검 중첩형 VFX를 조정했다.
  • 결과: 컴파일과 칼무리/성능/무기별 Echo Matrix QA가 모두 통과했다.

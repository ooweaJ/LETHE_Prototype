  • BeginRun(V1WeaponId)가 더 이상 시작 기억을 자동 지급하지 않습니다.
  • Unity 스모크 메뉴도 Start Dual Blades, Start Greatsword, M2 Loop 기준으로 정리했습니다.

3. 테스트 결과와 근거

  • dotnet build LETHE/Assembly-CSharp.csproj --nologo: 통과, legacy warning 7개, error 0개.

  • Start Dual Blades 스모크: weapon 절단쌍검, memories [], death False.

4. 결정한 것

첫 화면은 빌드 선택이 아니라 무기 선택으로 둡니다. 기억 선택은 첫 보상 카드가 담당하게 해서, 플레이어가 “왜 혈반이 무기에 붙어 있지?”라고 느끼지 않게 합니다.

5. 문제 또는 리스크

시작 직후 기억이 없으므로 첫 보상이 너무 늦게 느껴지면 다시 보상 속도를 봐야 합니다. 현재 XP 보정상 첫 보상은 빠르게 뜨는 편입니다.

6. GPT/Claude 인계 요약

Codex가 시작 UI를 4개 빌드 카드에서 2개 무기 카드로 되돌렸다. 칼무리/혈반은 첫 보상에서 선택되며, 첫 레벨업 카드에 두 기억이 정상 노출되는 것을 확인했다.

7. 다음 Codex 작업

  • 직접 플레이로 첫 보상까지의 시간이 자연스러운지 확인합니다.

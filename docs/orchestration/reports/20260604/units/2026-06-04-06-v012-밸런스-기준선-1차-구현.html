1. 현재 빌드 상태

  • LETHE HTML Alpha는 v0.12 첫 밸런스 기준선 구현 상태다.
  • v0.11의 기억 획득, 가중 망각, 무기 잔향, 무기 진화 구조는 유지했다.
  • 이번 패스는 새 기능 확장이 아니라 굶주린 칼무리, 적 스케일링, telemetry, 전술 집중 가중치, 피의 늪 폭주 위험을 먼저 정리했다.

2. 오늘 바뀐 것

  • 브라우저 라벨과 QA 기대 버전을 v0.12로 올렸다.
  • 굶주린 칼무리를 28 DPS * dt로 교정하고 가까운 타깃 우선 soft cap을 적용했다.
  • 일반 적 HP/피해에 시간+레벨 하이브리드 스케일링을 추가했다.
  • JSON 로그에 balance와 telemetry를 추가했다.
  • telemetry는 피해원별 누적 피해/DPS, 보스 TTK/집중 DPS, 레벨업 시각, 3슬롯 완성 시각, 적 스케일링, 5초 샘플을 남긴다.
  • 전술 집중의 망각 가중치 반영을 12배에서 3배로 낮췄다.
  • 피의 늪은 쌍검 30%, 대검 20% 발동률로 낮추고, 지속 시간/틱/피해를 줄였으며 최대 활성 장판 5개 제한을 넣었다.
  • docs/CODEX_STATUS.md, docs/NEXT_TASKS.md, docs/devlog/2026-06-04.md를 v0.12 기준으로 갱신했다.

# 2026-07-10-06 - 공통 잔향 본체 제거와 무기별 VFX 재분리

## 1. 현재 빌드 상태

`Dev_Prototype_v1`의 잔향 VFX가 다시 수정됐다. 이전 구현은 무기별 장식은 있었지만 공용 프롬프트, 링, 표식이 너무 크게 남아 있어서 단검과 대검이 같은 VFX처럼 보이는 문제가 있었다.

## 2. 오늘 바뀐 것

- `docs/무기별 잔향 및 기억.md`를 참고해 무기별 원칙을 다시 적용했다.
- 단검 계열은 짧고 빠른 바늘, 체인, 틱, 스택 컷 중심으로 정리했다.
- 대검 계열은 큰 판, 창, 시계장, 방패벽, 붕괴 구덩이 중심으로 정리했다.
- 공통 빨간 혈반 펄스, 처형 프롬프트/헤일로, 추적 락 링, 망각 프롬프트/링처럼 양쪽 무기를 비슷하게 보이게 만들던 본체 VFX를 제거하거나 약화했다.
- 주요 변경:
  - 혈반: 단검은 혈침 다발, 대검은 대검 혈 발도/반달 위주.
  - 균열: 단검은 바늘 분사와 미세 파문, 대검은 전방 균열.
  - 처형: 단검은 선고 바늘, 대검은 처형판/지면 문양/길로틴.
  - 추적: 단검은 부채꼴 추적침, 대검은 큰 창 예고.
  - 정지: 단검은 작은 시계 틱, 대검은 큰 시간장과 시계바늘.
  - 잿빛: 단검은 패리 바늘, 대검은 방패벽.
  - 망각: 단검은 스택 컷/점프 룬, 대검은 중력우물/크레이터.

## 3. 테스트 결과와 근거

- `dotnet build LETHE/Assembly-CSharp.csproj --nologo`: 성공, 기존 경고 7개, 오류 0개.
- Unity editor state: `Dev_Prototype_v1`, 포트 `7890`, 컴파일 중 아님.
- Unity compilation errors: `0`.
- Unity console errors: `0`.
- `LETHE/V1 QA/Echo Matrix Dual Blades`: MCP 메뉴 실행 성공.
- `LETHE/V1 QA/Echo Matrix Greatsword`: 1회 시도했지만 MCP가 `Error polling queue: fetch failed`를 반환했다. 이후 에디터 상태, 컴파일 에러, 콘솔 에러는 정상이다.

## 4. 결정한 것

잔향 VFX는 같은 계열 색이나 아이콘을 공유하더라도, 화면에서 처음 읽히는 실루엣은 무기별로 달라야 한다. 단검은 다단 잔상, 대검은 큰 판정 덩어리가 먼저 보여야 한다.

## 5. 문제 또는 리스크

이번 수정은 자동 검증상 문제는 없지만, 실제로 “똑같아 보인다”는 감각이 사라졌는지는 직접 플레이가 필요하다. 특히 혈반, 처형, 추적, 잿빛, 망각은 F12 `Echo One`으로 하나씩 확인해야 한다.

## 6. GPT/Claude 인계 요약

무기별 잔향 차별화는 단순 크기 조절이 아니라 공용 VFX 본체 제거 문제였다. 다음 리뷰는 각 잔향별 색보다 실루엣, 발동 위치, 타격 판정 언어가 무기 컨셉과 맞는지 봐야 한다.

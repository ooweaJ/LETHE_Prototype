# 2026-07-08-05 - 칼무리 콘셉트 프리뷰 식별성 분리

## 1. 현재 빌드 상태

`Dev_Prototype_v1`의 F12 `K1`~`K4` 칼무리 프리뷰가 다시 수정됐다. 첫 버전은 전부 비슷한 청록 칼날로 보여서 비교 도구로 부족했고, 이번 수정은 색과 형태부터 다르게 갈랐다.

## 2. 오늘 바뀐 것

- `K1`: 붉은/주황 상처 포식형. 상처 원반, 물어뜯는 링, 안쪽으로 수축하는 이빨, 붉은 검흔을 사용한다.
- `K2`: 파란 궤적 증식형. 긴 공격 리본 잔상과 지연 절단선을 사용한다.
- `K3`: 보라/흰 군집 교차참격형. 방사형으로 칼이 모이고 큰 X자 충격을 만든다.
- `K4`: 보라 표식 폭주형. 표식 링과 주변 적으로 갈라지는 링크/보조 표식을 사용한다.

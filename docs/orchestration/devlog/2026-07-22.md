## Blood Visibility Regression Fix / Remaining Echo VFX Plan

- Trigger:
  - jaewoo reported that Blood Echo appears once and then stops appearing.
  - jaewoo asked to organize the remaining Echo / VFX direction with examples.
- Diagnosis:
  - Greatsword Blood accent could be skipped when the base weapon hit killed the target before the Echo accent ran.
  - Dense Dual Blades still applied Blood marks, but recent dense VFX throttling suppressed almost all visible Blood read after the first obvious trigger.
  - There was also an impossible legacy branch guarded by `bloodLevel < 0`, so the fallback Blood pulse never ran.
- Applied:

  - Dense Dual Blades now spawns a lightweight repeated Blood mark read on the first allowed dense hit: pulse, two suture cuts, and a +5 bloom hint.
  - Removed the dead `bloodLevel < 0` fallback branch.
  - Generated and saved a remaining Echo / Ultimate VFX concept board at `docs/orchestration/evidence/2026-07-22-remaining-echo-vfx-concept-board.png`.

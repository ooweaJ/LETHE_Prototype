## Skill SFX Runtime Pass

- Jaewoo requested skill-appropriate sounds and asked to reference Vampire Survivors-like audio direction.
- Research takeaway:
  - survival auto-combat needs short, readable, repeatable sounds.
  - projectile impacts should be controlled because too many simultaneous hit sounds quickly become noisy.
  - retro/gameplay SFX can lean on square/noise/low sample-rate style signals, but LETHE should use original sounds rather than copied samples.
- Updated `LETHE/Assets/_dev/Scripts/PrototypeV1/V1GameManager.cs`:
  - added `V1SfxWave`.
  - replaced simple sine-only `MakeToneClip` with `MakeSfxClip`.
  - added deterministic pseudo-noise for impact/noise layers.
  - added replay throttling through `sfxLastPlayed` and `PlaySfx(id, volumeMul, minInterval)`.
  - generated original clips for weapons, Kalmuri, blood, utility memories, ultimates, pickups, kills, warnings, level-up, run end, and player hit.
  - wired sound calls to the relevant runtime trigger points.

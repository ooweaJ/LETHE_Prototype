# Review Responses

이 폴더는 Claude, GPT, Codex CLI 같은 검토 에이전트의 응답을 저장하는 현재 기준 위치다.

레거시 `docs/review_responses/`는 마이그레이션 전 기록으로만 본다. 새 응답은 이 폴더에 작성한다.

권장 파일명:

```text
docs/orchestration/review_responses/YYYY-MM-DD-topic-claude.md
docs/orchestration/review_responses/YYYY-MM-DD-topic-codex.md
docs/orchestration/review_responses/YYYY-MM-DD-topic-double-check.md
```

응답은 Discord 본문에 길게 붙이지 않는다. Discord에는 상태 요약과 첨부만 보내고, 전체 판단은 이 폴더의 Markdown을 기준으로 보존한다.

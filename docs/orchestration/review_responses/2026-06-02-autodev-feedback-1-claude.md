## 결론

- **GO_TO_HUMAN_TEST (조건부)**

---

## 이유

1. **AI 테스트 GO_CANDIDATE 달성.** Alpha Fun Score 0.8883, irritationRate 1.04%, regretRate 80.83%. 망각이 짜증보다 후회로 작동한다는 핵심 검증 항목이 봇 프록시 기준으로 충족됨.
2. **WP1 목표 범위 내 구현 완료.** buildIdentity hook, HUD 빌드명/시너지/의존 기억 표시, 태그 칩까지 요구 범위를 벗어나지 않음.
3. **restartRate 90%, immediateQuitRate 2.59%.** 초반 전투 루프가 플레이어를 망각 단계까지 끌고 가는 기본 조건을 만족함.
4. **`firstForgetUseAvgSec` 110초 — 90초 기준 초과.** UI에서 빌드 카드를 읽는 데 시간이 걸리고 있을 가능성. 사람 테스트 전 한 번 점검할 가치 있음.
5. **`멈춘 초침` 삭제율 7.25% (최저, 다음 순위 대비 절반).** 이 기억이 지나치게 강하거나 삭제하기 싫은 특이 케이스일 수 있음. 밸런스 문제가 아닐 수도 있지만 사람 테스트에서 관찰 필요.

## 테스트 기준

- **AI 테스트 기준:** 텍스트 압축 후 `npm run ai:test:quick`에서 confusionRate 1.55% 이하 유지, firstForgetUseAvgSec 90초 이하 확인.
- **사람 플레이테스트 관찰 기준:**
  - 망각 화면에서 "왜 이 기억이 사라지나" 질문 없이 넘어가는지.
  - 빌드 이름을 보고 "내가 뭘 하고 있는지 알겠다"는 반응이 있는지.
  - `멈춘 초침` 기억을 플레이어가 의도적으로 지키는지, 아니면 그냥 선택지에서 피하는지.

# LETHE PRD

## Purpose

이 문서는 LETHE가 무엇을 만들고 왜 만드는지를 고정하는 제품 기준이다. 세부 구현, 클래스 구조, 파일명, 테스트 명령은 `docs/TECH.md`, `docs/TASK.md`, `docs/TEST.md`에서 관리한다.

## Product Summary

LETHE는 기억을 얻고, 강화하고, 잃고, 잔향으로 다시 조립하는 2D 액션 로그라이트다. 핵심 감정은 "강해진 것을 잃는 아쉬움"과 "잃은 것이 다른 형태로 남아 다시 빌드가 되는 쾌감"이다.

## Core Experience

- 플레이어는 전투 중 기억을 얻고 강화한다.
- 가장 강한 기억은 망각 후보가 되어, 성장 자체가 리스크가 된다.
- 망각된 기억은 약한 복사본이 아니라 형태가 바뀐 잔향으로 남는다.
- 같은 기억을 다시 얻으면 기존 잔향과 공명해 "돌아왔다"는 감각을 준다.
- 잔향 +5와 잔향 조합 궁극은 후반 뽕맛의 핵심 보상이다.

## MVP Scope

- Unity `Dev_Prototype_v1`에서 평가 가능한 코어 프로토타입을 만든다.
- 무기 2종: 쌍검, 대검.
- 기억 8종.
- 잔향 8종.
- 궁극 잔향 4종.
- 실제 플레이 M2 루프: 전투 -> XP -> 카드 선택 -> 기억 강화 -> 망각 -> 잔향 -> 공명 -> +5 -> 궁극.
- 판독 가능한 플레이어, 적, 무기, 기억, 잔향, 궁극 VFX.

## First Prototype Success

첫 목표는 HTML보다 더 명확한 Unity 코어 프로토타입이다. 플레이어가 아래를 직접 느낄 수 있어야 한다.

- 쌍검은 빠르고 연쇄적인 무기다.
- 대검은 느리고 크고 묵직한 무기다.
- 기억과 잔향은 화면에서 다른 형태로 읽힌다.
- 잔향은 무기별로 다르게 반응한다.
- 망각이 짜증이 아니라 아쉬움과 빌드 전환으로 느껴진다.
- 궁극은 단순 수치 증가가 아니라 화면과 리듬이 바뀌는 보상이다.

# LETHE v0.5 core-fun 기획 검토 요청 - 2026-06-02

## 대상

Claude Code에게 초반 재미, 로그라이크/뱀서식 성장감, 사람 테스트 전 핵심 보완 방향을 묻는다.

## 현재 상황

- 현재 빌드: LETHE HTML alpha v0.4 human-test candidate.
- 핵심 루프: 기억 3개 선택 → 전투 → 보스 처치 → 사라질 기억 예측 → 의존도 기반 기억 상실 → 잔향 보상 → Q1/Q2/Q3 설문 → JSON 로그 다운로드.
- 구현 범위: 무기 2종, 기억 6종, 활성 기억 슬롯 3개, 적 4종, 3페이즈 보스 1종.
- 원래 목표: 망각이 짜증이 아니라 후회로 느껴지는지 검증한다.
- 새 사용자 피드백: 지금은 기억/망각 이전에 게임이 너무 루즈하다. 초반 재미가 약하고, 로그라이크 맛이 부족하며, Vampire Survivors처럼 적이 계속 몰려오는 압박과 처치 후 스탯/성장 선택이 없다. 사람 테스트 전에 이 문제를 먼저 보완해야 한다.
- 범위 제한: 메타 진행, 상점, 최종 보스, 6개 초과 기억, 다중 지역 구조는 아직 추가하지 않는다.

## v0.2-v0.4에서 바뀐 것

- 첫 망각 타이밍을 평균 9분 근처로 늦췄다.
- 기본 실험값을 `echo=0.50`, `ui=0.78`로 설정했다.
- `처형자의 섬광` 삭제 쏠림을 낮추고 삭제 weight를 로그에 남기게 했다.
- 망각 결과 화면에서 `사라진 행동`과 `잔향 변형`을 분리해 표시한다.
- Q3 자유응답으로 “방금 사라진 기억 이름을 떠올릴 수 있는지”를 추가했다.
- 5-8명용 human playtest guide를 작성했다.
- 기억 발동명, 피해 숫자, 스파크, 투사체 trail, 보스 impact, 의존도 UI를 추가했다.

## 문제 진단

- AI 지표는 망각 루프 기준으로 좋지만, 실제 플레이어 피드백은 “기억이고 뭐고 초반이 루즈하다”에 가깝다.
- 초반 3-5분에 적이 충분히 몰려오지 않고, 처치 리듬과 위험/쾌감이 약하다.
- 런 중 성장 선택이 없어 로그라이크/뱀서식 기대감이 약하다.
- 지금은 망각 감정 검증보다 “첫 플레이가 재미있어서 망각까지 가는가”를 먼저 해결해야 한다.

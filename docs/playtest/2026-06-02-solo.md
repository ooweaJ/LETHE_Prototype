## Stop Rules

- If post-loss mob clearing is still `impossible` or `too hard`, start a v0.7.1 balance loop.
- If mob clearing is manageable but echoes are not visible, do a display-only v0.7.1 visibility pass.
- If the loss feels like `no big deal`, do not add weapons yet; first decide whether echo visibility or baseline weapon strength caused it.
- If the run makes you want another try, keep v0.7 as the first solo-test accepted candidate.

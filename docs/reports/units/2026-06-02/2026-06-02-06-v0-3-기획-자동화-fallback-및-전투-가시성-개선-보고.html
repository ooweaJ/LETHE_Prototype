2026-06-02-06 - v0.3 기획 자동화 fallback 및 전투 가시성 개선 보고

1. 현재 빌드 상태

  • LETHE HTML alpha는 v0.3 pre-human-test polish 후보 상태다.
  • AI 1000런 기준 verdict는 계속 GO_CANDIDATE다.
  • 사람 테스트는 아직 진행하지 않는다. 전투 연출과 시스템 가시성을 먼저 더 끌어올리는 단계다.

2. 오늘 바뀐 것

  • scripts/ask_codex_review.js를 추가해 Codex CLI로도 같은 기획 답변을 파일로 받을 수 있게 했다.
  • npm run review:codex와 npm run review:codex:dry를 추가했다.
  • 브라우저 표기를 v0.3으로 올렸다.
  • 기억 발동 시 기억 이름이 전투 화면에 플로팅 텍스트로 뜨게 했다.
  • 피해 숫자, 타격 스파크, 투사체 trail, 보스 등장/페이즈 impact를 추가했다.
  • 기억 슬롯에 레테의 시선 태그와 의존도 퍼센트를 표시해 어떤 기억이 삭제 위험에 가까운지 더 잘 보이게 했다.

3. 테스트 결과와 근거

  • node --check src/game.js: 통과.
  • node --check scripts/ask_codex_review.js: 통과.
  • npm run review:codex:dry: 최신 v0.3 프롬프트와 출력 경로 확인.
  • npm run ai:test:quick: GO_CANDIDATE, Alpha Fun Score 0.7798.
  • npm run ai:test: GO_CANDIDATE, Alpha Fun Score 0.7737.
  • 브라우저 QA:
  • HTML Alpha Prototype v0.3 표시 확인.
  • ?qa=fast에서 질문 화면과 결과 화면까지 도달했다.
  • 레테의 시선 태그와 의존도 퍼센트가 기억 슬롯에 표시됐다.
  • 결과 패널은 내부 스크롤 없이 표시됐다.
  • JSON payload에 memoryNames, questionNames, forgottenName, deletionWeights, survey, echo, experiment가 들어갔다.
  • payload의 experiment.version은 v0.3으로 기록됐다.

4. 결정 사항

  • Claude Code는 설치됐지만 로컬 로그인/승인이 필요하므로, 내일 로컬 Mac에서 claude auth login을 진행해야 한다.
  • Claude가 막힐 때는 먼저 Codex CLI fallback을 사용한다.
  • 외부 기획 검토가 막혀도, 현재 범위에서는 새 콘텐츠를 추가하지 않고 전투 가시성/시스템 전달력만 개선한다.

5. 문제 또는 리스크

  • Codex CLI 방식도 로컬 Codex 로그인 세션과 플랜 사용 한도에 의존한다.
  • Claude Code는 설치됐지만 인증 흐름에서 멈추므로, 원격으로는 브라우저 승인까지 진행할 수 없다.
  • 공격 연출은 개선됐지만, 잔향이 “상실 후 변형”처럼 느껴지는지는 아직 결과 화면 중심이라 추가 개선 여지가 있다.

6. GPT/Claude 인계 요약

  • v0.3/version-up 기획 질문 Markdown은 준비되어 있다.
  • Claude 실행은 로컬 로그인 후 가능하다.
  • Codex CLI 실행은 npm run review:codex로 가능하며, 답변은 docs/review_responses/YYYY-MM-DD-codex.md에 저장된다.

7. 다음 Codex 작업

  • 잔향 보상이 결과 화면뿐 아니라 다음 플레이 감각으로 더 잘 이어지게 만든다.
  • 공격 이펙트가 기억별로 더 구분되는지 브라우저에서 계속 확인한다.
  • 사람 테스트는 v0.3 polish가 충분히 보인 뒤 docs/HUMAN_PLAYTEST_GUIDE.md로 진행한다.

8. 포트폴리오 메모

  • 문제: Claude 인증이 막히면 기획 자동화가 멈추고, v0.2는 기능은 통과했지만 전투 체감이 약했다.
  • 방향: Claude가 막힐 때 Codex CLI fallback으로 같은 프롬프트를 검토하고, 사람 테스트 전 핵심 전투 체감을 강화한다.
  • 실행: Codex CLI 검토 스크립트와 v0.3 전투 가시성 개선을 추가하고 브라우저 QA로 회귀를 확인했다.
  • 결과: v0.3 프로토타입은 기억 의존도와 망각 위험이 더 눈에 띄는 상태가 됐다.

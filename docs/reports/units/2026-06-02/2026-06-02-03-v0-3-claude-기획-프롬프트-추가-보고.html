2026-06-02-03 - v0.3 Claude 기획 프롬프트 추가 보고

1. 현재 빌드 상태

  • LETHE v0.2는 AI 기준 human-test candidate 상태를 유지한다.
  • v0.3/version-up 판단은 아직 실행된 Claude 답변이 아니라, 새 검토 프롬프트 준비 단계다.

2. 오늘 바뀐 것

  • docs/review_prompts/2026-06-02.md를 추가했다.
  • 프롬프트는 v0.3를 사람 테스트 전에 시작할지, 아니면 v0.2 QA 후 사람 테스트로 갈지 묻는다.
  • 프롬프트는 콘텐츠 추가 없이 감정선, 결과 피드백, 테스트 질문만 좁게 검토하도록 제한한다.

3. 테스트 결과와 근거

  • npm run review:claude:dry: 최신 프롬프트로 docs/review_prompts/2026-06-02.md 선택 확인.
  • 예상 출력 경로: docs/review_responses/2026-06-02-claude.md.
  • npm run review:claude: 실패. 현재 로컬 셸에서 claude 실행 파일을 찾지 못했다.

4. 결정 사항

  • Claude Code 자동 검토 흐름은 유지한다.
  • 실제 Claude 호출은 Claude Code가 설치되어 있거나 PATH에 잡힌 환경에서 다시 실행한다.

5. 문제 또는 리스크

  • 현재 로컬 환경의 PATH에는 claude가 없다.
  • 아직 Claude v0.3 답변 파일은 생성되지 않았다.

# 2026-06-02-17 - AI 협업 포트폴리오 문서 및 로컬 doctor 보고

## 1. 현재 빌드 상태

- LETHE HTML alpha v0.5는 사람 테스트 준비 상태를 유지한다.
- 이번 작업은 게임 기능 추가가 아니라 AI 협업 증거와 로컬 재현성 검증 라인을 추가한 것이다.

## 2. 오늘 바뀐 것

- `docs/ai/README.md`를 추가했다.
- `docs/ai/ai-collaboration.md`에 AI 역할, workflow, evidence rules, prompt rules, developer control rules를 정리했다.
- `docs/ai/ai-failure-cases.md`에 외부 전송 차단, Browser plugin 실패, Claude 역할 재정의, OpenAI fallback 제거 사례를 기록했다.
- `docs/adr/ADR-001-html-prototype-before-unity.md`를 추가했다.
- `docs/adr/ADR-002-test-result-planning-pipeline.md`를 추가했다.
- `docs/portfolio/LETHE-case-study.md`를 추가했다.
- `scripts/check_local_pipeline.js`를 추가했다.
- `npm run doctor`, `npm run doctor:deep` 명령을 추가했다.

## 3. 테스트 결과와 근거

- `node --check scripts/check_local_pipeline.js`: 통과.
- package.json JSON parse 검사: 통과.
- `npm run doctor`: 24 pass, 0 warn, 0 fail.
- `npm run doctor:deep`: 30 pass, 0 warn, 0 fail.
- doctor는 Node 버전, 필수 npm scripts, AI 협업 문서/ADR/portfolio 문서의 필수 섹션, `claude`/`codex`/`git`/`npm` 명령, `.env`/Discord 설정, 최신 프롬프트 존재 여부를 확인한다.
- doctor deep은 외부 전송 없는 dry-run과 문법 검사를 추가로 실행한다.

## 4. 결정 사항

- 모든 AI 대화 원문을 저장하지 않는다.
- 설계가 바뀐 순간, AI/자동화가 실패한 순간, 개발자가 AI 제안을 수정한 순간, 테스트 결과가 방향을 바꾼 순간만 문서화한다.
- 다른 로컬에서 시작할 때는 먼저 `npm run doctor`를 실행한다.
- 장시간 무인 작업 전에는 `npm run doctor:deep`으로 dry-run 상태를 확인한다.

## 5. 문제 또는 리스크

- `claude`나 `codex` CLI가 없는 로컬에서는 doctor가 warning을 낸다.
- Discord webhook은 `.env`가 없거나 `DISCORD_WEBHOOK_URL`이 없으면 warning만 낸다.
- 외부 전송과 push는 doctor가 대신 승인하지 않는다.

## 6. GPT/Claude 인계 요약

- 이 프로젝트는 AI를 코드 생성기로만 쓰지 않는다.
- Claude/GPT는 테스트 결과 기반 기획 파트너이며, Codex는 구현/검증/기록 담당이다.
- 실패 사례와 ADR은 포트폴리오에서 “AI가 틀린 것을 개발자가 통제했다”는 근거로 사용한다.

## 7. 다음 Codex 작업

- `npm run doctor`와 `npm run doctor:deep`을 실행해 로컬 재현성 체크를 검증한다.
- 사람 테스트 후 결과 요약을 `docs/ai/ai-collaboration.md`와 portfolio case study에 반영한다.

## 8. 포트폴리오 메모

- 문제: AI를 썼다는 사실만으로는 포트폴리오 차별점이 약하다.
- 방향: 문제 정의, 제약, AI 제안, AI 실패, 개발자 수정, 검증 결과를 문서화한다.
- 실행: AI 협업 문서, failure cases, ADR, case study, local doctor를 추가했다.
- 결과: LETHE는 “AI로 만든 게임”보다 “AI를 통제해 프로토타입 검증 파이프라인을 만든 프로젝트”로 설명할 수 있게 됐다.

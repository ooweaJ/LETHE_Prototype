- 단, 설계 문제가 아니라 실제 브라우저 검증이 아직 없어서 보류라는 판단이다.

## 2. 오늘 바뀐 것

- `npm run review:claude:dry`: v0.5 평가 프롬프트 선택 확인.

## 4. 결정 사항

- 지금은 새 기능을 더 만들지 않는다.
- `post-forgetting power drop` 28%는 지금 보정하지 않고 사람 테스트에서 체감 확인한다.
- 기억별 투자-망각 상호작용은 v0.6 후보로 보류한다.
- 사람 테스트 전 남은 gate는 v0.5 브라우저 동작 검증이다.

## 5. 문제 또는 리스크

- 이 Codex 세션에서는 Browser 플러그인이 Windows sandbox 오류로 열리지 않았다.
- 실제 플레이어가 만질 레벨업 화면이 깨져 있으면 사람 테스트 전체가 무효가 된다.

## 6. GPT/Claude 인계 요약

- Claude는 `GO_TO_HUMAN_TEST`가 아니라 `ITERATE_BEFORE_TEST`를 선택했다.
- 이유는 설계 부족이 아니라 브라우저 동작 검증 미완이다.
- 검증 통과 시 바로 사람 테스트로 가도 된다는 판단이다.

## 7. 다음 Codex 작업

- 한 런 완료 후 JSON `runGrowth`가 실제 선택과 일치하는지 확인한다.

- 방향: 새 기능 추가가 아니라 사람 테스트 전 실행 신뢰성을 확보한다.

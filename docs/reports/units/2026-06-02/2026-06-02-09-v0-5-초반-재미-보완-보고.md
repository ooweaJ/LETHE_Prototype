# 2026-06-02-09 - v0.5 초반 재미 보완 보고

## 1. 현재 빌드 상태

- LETHE HTML alpha는 v0.5 core-fun human-test candidate 상태다.
- 기존 v0.4는 망각/잔향 전달은 좋아졌지만, 실제 플레이 피드백상 초반이 루즈했다.
- v0.5는 사람 테스트 전에 “재미있어서 망각까지 가는가”를 먼저 보완한 버전이다.

## 2. 오늘 바뀐 것

- `docs/review_prompts/2026-06-02.md`를 초반 재미/로그라이크 맛 중심으로 갱신했다.
- 실제 Claude 자동 호출은 외부 프롬프트 전송 정책으로 차단됐다.
- 초반 적 스폰을 더 촘촘한 pack 단위로 바꿨다.
- 적 처치 XP와 런 중 레벨업 3지선다 스탯 선택을 추가했다.
- 레벨업 선택은 메타 진행이 아니라 해당 런 안에서만 유지된다.
- 성장 선택은 피해, 공격 속도/쿨다운, 범위/넉백, 생존, XP 획득으로 제한했다.
- JSON payload에 `runGrowth`를 추가했다.
- AI test에 `Early Fun Score`, `earlyKillTempo`, `earlyLevelUps`를 추가했다.

## 4. 결정 사항

- 사람 테스트 전 우선순위는 망각 문구가 아니라 초반 재미와 성장 리듬이다.
- 새 기억, 상점, 메타 성장, 다중 지역은 여전히 만들지 않는다.
- 현재 성장은 “런 중 임시 스탯 선택”까지만 허용한다.

## 5. 문제 또는 리스크

- Browser 플러그인 QA는 Windows sandbox startup 오류로 실행하지 못했다.
- 망각 직후 전투력 딥은 `27.8-28.0%`로 30% 목표보다 낮다.
- 초반 재미 지표는 AI 기준 좋아졌지만, 실제 사람이 레벨업 선택을 충분히 로그라이크답게 느끼는지는 확인해야 한다.

## 6. GPT/Claude 인계 요약

- Claude 실제 호출은 이 세션에서 차단됐다.
- 다음 기획 검토에는 “초반 재미가 살아났는지”와 “망각 감정 검증으로 돌아가도 되는지”를 같이 물어야 한다.
- 사람 테스트 후 대표 발화와 JSON `runGrowth`를 함께 넘긴다.

2026-06-02-07 - v0.4 사람 테스트 후보 보고

1. 현재 빌드 상태

  • LETHE HTML alpha는 v0.4 human-test candidate 상태다.
  • AI 1000런과 heavy 5000런/3스테이지 모두 GO_CANDIDATE다.
  • 다음 큰 단계는 더 많은 기능 추가가 아니라 5-8명 사람 테스트다.

2. 오늘 바뀐 것

  • 브라우저 표기를 v0.4로 올렸다.
  • 기본 AI 테스트의 UI 선명도 값을 ui=0.78로 올렸다.
  • 결과 화면에서 사라진 행동과 잔향 변형을 분리했다.
  • 잔향 변형 카드에 요약 문장과 stat badge를 추가했다.
  • JSON payload에 echoTransformation을 추가했다.
  • AI 테스트 기본 명령과 alpha test 문서 예시를 echo=0.50, ui=0.78 기준으로 맞췄다.

3. 테스트 결과와 근거

  • node --check src/game.js: 통과.
  • node --check alpha_test/src/config.js: 통과.
  • node --check alpha_test/src/run_alpha.js: 통과.
  • npm run ai:test:quick: GO_CANDIDATE, Alpha Fun Score 0.8190.
  • npm run ai:test: GO_CANDIDATE, Alpha Fun Score 0.8261, regret 85.6%, irritation 0.4%, prediction 85.8%.
  • npm run ai:test:heavy: GO_CANDIDATE, Alpha Fun Score 0.8369, regret 87.6%, irritation 0.3%, prediction 87.7%.
  • npm run ai:sweep: echo=0.50, ui=0.78이 현재 가장 안정적인 GO_CANDIDATE 프리셋임을 확인했다.
  • 브라우저 QA:
  • HTML Alpha Prototype v0.4 표시 확인.
  • ?qa=fast에서 질문 화면과 결과 화면까지 도달했다.
  • 잔향 변형 카드와 echo badge가 표시됐다.
  • 결과 패널은 내부 스크롤 없이 표시됐다.
  • JSON payload에 echoTransformation, uiClarity: 0.78, experiment.version: v0.4가 들어갔다.

4. 결정 사항

  • 지금부터는 Codex가 더 큰 콘텐츠를 추가하기보다 사람 테스트를 위한 안정화 상태로 본다.
  • 상점, 메타 진행, 최종 보스, 기억 추가, 다중 지역 구조는 여전히 만들지 않는다.
  • 남은 29.4-29.6% power drop soft warning은 코드로 과보정하기보다 사람 테스트에서 실제 체감으로 확인한다.

5. 문제 또는 리스크

  • Alpha Fun Score는 올랐지만 목표 0.89+에는 아직 못 미친다.
  • 망각 직후 전투력 딥은 목표 30%보다 아주 조금 낮다.
  • AI 지표가 좋아져도 실제 사람이 잔향을 “상실 후 변형”으로 느낄지는 테스트가 필요하다.

6. GPT/Claude 인계 요약

  • 외부 기획 검토 없이도 v0.4는 사람 테스트 후보까지 도달했다.
  • Claude/GPT에게 다음에 물을 질문은 “사람 테스트 결과를 어떻게 해석해 v0.5 방향을 잡을지”다.
  • Codex CLI fallback은 준비되어 있지만, 최종 기획 판단은 사람 테스트 후 GPT/Claude에 다시 묻는 것이 좋다.

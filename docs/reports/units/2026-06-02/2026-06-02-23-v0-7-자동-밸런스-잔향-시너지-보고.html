2026-06-02-23 - v0.7 자동 밸런스/잔향 시너지 보고

1. 현재 빌드 상태

  • LETHE HTML alpha는 v0.7 weapon/echo balance solo-test candidate 상태다.
  • v0.6 1인 체감 피드백에서 나온 “기억을 잃은 뒤 너무 약해져 쫄몹을 못 잡는다” 문제를 자동 밸런스 루프 입력으로 삼았다.
  • 넓은 사람 테스트가 아니라, 사용자 1인 체감 테스트로 다시 확인할 단계다.

  • 굶주린 칼무리: 무기 출혈 잔향.

  • 사이드 패널 잔향 목록에 무기 잔향 라벨을 추가했다.
  • AI 시뮬레이터의 무기 baseDps와 weapon residue echo proxy를 v0.7 방향에 맞췄다.
  • v0.7 결과 판단 프롬프트 docs/review_prompts/2026-06-02-v07-balance-eval.md를 추가했다.

4. 결정 사항

  • v0.7은 로컬 Codex 판단 기준 GO_TO_SOLO_PLAYTEST_CANDIDATE다.
  • 신규 무기/무기 특수기/상점/메타 진행은 아직 넣지 않았다.
  • 먼저 기존 2무기와 잃은 기억의 무기 잔향 시너지로 결손 구간이 버틸 만해졌는지 확인한다.
  • 1인 테스트에서 여전히 쫄몹 처리가 무너지면 v0.7.1 자동 밸런스 루프를 실행한다.

5. 문제 또는 리스크

  • AI 지표상 초반 처치 템포가 크게 좋아졌지만, 실제로 너무 쉬워졌는지는 모른다.
  • Echo pivot score는 v0.6보다 살짝 낮아졌다. 무기 바닥 성능이 문제를 해결한 대신, 잔향 변형의 체감이 약해질 위험이 있다.
  • 예측율은 여전히 높다.
  • 무기 특수기와 신규 무기는 아직 기획 후보일 뿐 구현하지 않았다.

6. GPT/Claude 인계 요약

  • v0.7 결과 판단 프롬프트는 준비됐다: docs/review_prompts/2026-06-02-v07-balance-eval.md.
  • 현재 Codex 세션에서는 프로젝트 요약의 외부 Claude 전송이 막히므로, 필요하면 사용자의 신뢰된 로컬 터미널에서 실행한다.
  • 이번 자동 루프에서는 로컬 Codex 판단으로 v0.7을 1인 체감 테스트 후보로 둔다.

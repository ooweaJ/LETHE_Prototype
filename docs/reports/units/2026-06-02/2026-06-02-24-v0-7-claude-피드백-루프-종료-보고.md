- Claude가 이전에 안 됐던 이유는 인증/외부 전송 정책 문제였다. 이번에는 `npm run autopilot:preflight` 통과 후 실제 Claude 호출까지 성공했다.
- Echo pivot score 하락은 아직 체감 검증 전이라 수치 보정으로 해결하지 않는다.
